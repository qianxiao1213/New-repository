# 赛博朋克2077 + DG-Lab 整合控制软件

## 一、软件简介

本软件可以实时读取《赛博朋克2077》游戏中的角色状态（生命值、体力、战斗状态等），并根据游戏状态自动控制 DG-Lab 电击设备的强度，实现游戏与设备的联动。

## 二、系统要求

- Windows 10 / Windows 11
- 已安装《赛博朋克2077》游戏
- 已安装 Cyber Engine Tweaks (CET) 模组
- DG-Lab 电击设备及配套 App

## 三、安装步骤

### 1. 安装游戏模组

将以下两个模组放入游戏目录：
Cyberpunk 2077\bin\x64\plugins\cyber_engine_tweaks\mods
├── PosLogger\ # Rezo Agwe 模组（输出游戏状态）
└── Test\ # 生命值监控模组（输出生命值/体力）


### 2. 运行软件

双击 `CyberpunkDGLab.exe` 启动程序。

### 3. 连接 DG-Lab

1. 打开 DG-Lab App
2. 点击连接 → 扫描二维码
3. 二维码会在软件界面中显示

## 五、使用方法

### 1. 首次使用

1. 启动游戏，加载存档
2. 运行本软件
3. 在"游戏监控"页面，点击"浏览"按钮设置：
   - PosLogger 目录：选择 `mods\PosLogger` 文件夹
   - 生命值文件：选择 `mods\Test\health_stamina_status.json`
4. 切换到"DGLab控制"页面，扫描二维码连接设备

### 2. 自动强度模式

连接成功后，软件会根据游戏状态自动调整强度：

- **非战斗状态**：强度 = 非战斗基础强度
- **战斗中**：强度 = 战斗基础强度 + (1 - 生命值百分比) × 权重系数
- **死亡或体力耗尽**：强度拉满（持续设定秒数）

### 3. 手动微调

在 App 上按按钮可以微调强度（叠加到自动强度上）：

| 按钮 | 功能 |
|------|------|
| A2 | A通道强度 +1 |
| A3 | A通道强度 -1 |
| B2 | B通道强度 +1 |
| B3 | B通道强度 -1 |
| A1 | A通道切换波形 |
| B1 | B通道切换波形 |

### 4. 通道选择

- **仅A通道**：只控制 A 通道，B 通道关闭
- **仅B通道**：只控制 B 通道，A 通道关闭
- **A/B通道**：两个通道同时控制

## 六、常见问题

### Q1: 软件无法连接设备？

1. 确保 DG-Lab App 已打开
2. 点击"刷新状态"查看连接状态
3. 重新扫描二维码连接

### Q2: 游戏数据显示不正常？

1. 检查游戏是否已加载存档
2. 检查 CET 模组是否正确安装
3. 在"游戏监控"页面重新选择文件路径

### Q3: 强度不变化？

1. 检查"监测生命值变化"是否勾选
2. 检查是否在稳定期内（连接后5秒）
3. 查看终端调试输出（勾选"终端调试输出"）

### Q4: App 闪退？

1. 检查强度参数是否超过设备上限
2. 重新连接设备
3. 重启软件
